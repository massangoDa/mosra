# MassCRM（Nuxt.js 顧客管理サービス）

![MassCRM UI](nuxt_fro/bro/public/img.png)

このプロジェクトは **Nuxt.js 3** をベースに構築された CRM（顧客管理）アプリケーションです。  
主にクライアント管理・請求書の作成と管理・カレンダーイベント操作など、ビジネス向けの基本的な業務を支援します。

## 🧭 概要

このアプリケーションの主な目的は以下の通りです。

- **顧客情報の管理**  
  顧客データの一覧・登録・編集に対応し、取引履歴や請求情報を閲覧可能。
- **請求書の作成と管理**  
  顧客ごとの請求書を生成・閲覧・更新。
- **カレンダーイベントの管理**  
  タスクやスケジュールを視覚的に操作できる。
- **インタラクティブなUI**  
  ユーザー体験を重視した、シンプルかつ直感的なインターフェース。

## 🗂 フォルダ構造

主要なディレクトリ構成は以下の通りです：
```aiignore
bro/
├── app/
│   ├── api/           # サーバーサイド用APIモジュール
│   ├── assets/        # CSSや画像などの静的アセット
│   ├── components/    # Vueコンポーネント
│   ├── composables/   # タイムフォーマットやユーザー情報管理ロジック
│   ├── layouts/       # ページのレイアウト
│   ├── pages/         # 各種ページ（Vueファイルベースのルーティング）
│   ├── plugins/       # カスタムプラグイン（oh-vue-icon、toastなど）
│   └── types/         # 型定義
├── public/            # 静的ファイル（faviconや画像データ等）
├── package.json       # プロジェクトの依存関係
├── nuxt.config.ts     # Nuxtの設定
└── tsconfig.json      # TypeScriptの設定
```

## ⚙️ 使用技術

- Vue 3
- Nuxt.js 3
- Vue Router

- axios（API通信）

### バックエンド
- Express.js
- MongoDB

### その他
- TypeScript

---

## 🚀 主な機能

1. **ユーザー認証**
    - ログイン・新規登録
    - JWTによるセキュアなアクセス制御

2. **顧客管理**
    - 顧客情報の登録・更新・削除
    - 顧客ごとの取引履歴を閲覧

3. **請求書管理**
    - 新規請求書の作成
    - 過去の請求書の管理・更新

4. **カレンダー機能**
    - イベントの追加・削除・一覧表示
